### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/shreyash-sharma/follow-me-eyes.git
   ```
3. Check emscripten version:
   ```sh
   pip3 install -r requirements.txt
   ```
4. Build WASM files using emscripten
   ```sh
   python3 server.py
   ```
5. Navigate to http://localhost:7000/
